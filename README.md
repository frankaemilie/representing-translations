# representing-translations

## Name
representing translations

## Description
This code will represent, manage and merge information from several translation lexicons. 

## Method
To test the programme, 3 results files are provided. These results were obtained by two different tools (Anymalign and GIZA++), applied to a corpus of parallel French-English sentences extracted from the Tatoeba website (French will be taken as the source language).
